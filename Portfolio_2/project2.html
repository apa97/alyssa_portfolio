UX/UI Design & App Development

Munchie is a voice-baased mobile application designed to help users....

Role: UX/UI Designer
Tools: Figma, React Native
Focus: Mental wellness
Year: 2025

Identifying the Problem

Many mindfulness apps overwhelm users with notifications and complex tracking systems, making it harder to maintain consistent habits.

Design Approach

The app focuses on simplicity and emotional comfort. The interface avoids pressure, encouraging users to engage naturally.

Development & Implementation

Built using React Native, focusing on smooth interactions and a clean UI structure.

Outcome

The final product provides a calm and supportive experience for users building mindfulness habits.

Challenges

Balancing minimal design while still providing meaningful functionality was a key challenge.

Reflection

This project strengthened my ability to design with empathy and focus on user-centered experiences.
